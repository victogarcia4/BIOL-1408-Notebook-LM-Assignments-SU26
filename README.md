# BIOL 1408 NotebookLM Assignments Dashboard

A dynamically synced dashboard and tracker for the **BIOL 1408** biology course, designed and curated by **Dr. Víctor García Martínez**.

This application tracks active learning objectives and integrates direct links to personalized **NotebookLM** study materials for students.

## Features

- **Dynamic NotebookLM Sync**: Automatically processes learning objectives and matches them with authorized NotebookLM resource links.
- **Bi-directional Integration**: Direct sync with Google Docs to pull the latest coursework parameters seamlessly.
- **Clean Unit Navigation**: Categorizes objectives by biology units (Unit 1: Chemistry & Cells, Unit 2: Energetics, Unit 3: Genetics, Unit 4: Molecular Regulation) for focused study.
- **Student Progress Companion**: Allows students to easily bookmark key materials and view their alignment.

## Tech Stack

- **Frontend**: React (with Vite, Tailwind CSS, Lucide icons, and Motion animations)
- **Backend**: Express (Node.js API proxy for secure sync operations)
- **Data Hydration**: Direct integration with Google Docs URLs
